# IntoTheForest

[![GITHUB](https://github.com/weiyunghsu/IntoTheForest/blob/85f4e4c73302eaf35881e61f2b442cbd6de504bb/screenshot/Google_Play_Store_badge320.png "title image")](https://play.google.com/store/apps/details?id=com.weiyung.intotheforest) Download Here! 

This app helps users know and share more the information about hiking in the forest. It provides features including: reading, sharing, collecting and publishing articles, and looking for the recommend routes on the map, also providing weather information, and help you find the way by messaging and calling.

## Flows
 - Import Google Login SDK with Firebase Authentication for user management
<img src="https://github.com/weiyunghsu/IntoTheForest/blob/50fc87bc81b8d55cfb344a8b1418635add419c67/screenshot/login.jpg" width="250" height="500" alt="登入"/>

 - In home page you can see all the articles other users post, and it will renew with livedata.
 - You can click one of the article to see its detail.
 - In detail page you can see the article and the gallery, also use the share button link to other apps
 - You can add the article you like, it will collect in favorite page

<img src="https://github.com/weiyunghsu/IntoTheForest/blob/50fc87bc81b8d55cfb344a8b1418635add419c67/screenshot/home.jpg" width="250" height="500" alt="首頁"/> <img src="https://github.com/weiyunghsu/IntoTheForest/blob/50fc87bc81b8d55cfb344a8b1418635add419c67/screenshot/detail.jpg" width="250" height="500" alt="詳細資訊"/> <img src="https://github.com/weiyunghsu/IntoTheForest/blob/50fc87bc81b8d55cfb344a8b1418635add419c67/screenshot/favorite.jpg" width="250" height="500" alt="最愛"/><br/>

 - You can see the map from detail page or bottomNavigation
 - In the map, you can choose the routes by spinner on the top
 - You can see the whole route, it contains the start point, end point, and other special stops in the middle.
 - You can link to the Central Weather Bureau Website by the weather button

<img src="https://github.com/weiyunghsu/IntoTheForest/blob/50fc87bc81b8d55cfb344a8b1418635add419c67/screenshot/map.jpg" width="250" height="500" alt="地圖"/> <img src="https://github.com/weiyunghsu/IntoTheForest/blob/50fc87bc81b8d55cfb344a8b1418635add419c67/screenshot/weather.jpg" width="250" height="500" alt="天氣"/>

 - When you get lost, you can call for help by the phone button.
 - If you have bad phone signal, you can get the location by location button, and input the SOS messenge, send it to us, we will help you immediately when we get the information.
 - You can post your jounery as a new article in the publish page, it contains start date, end date, texts and the picture in your phone gallery.
